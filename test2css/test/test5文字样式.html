文字样式

  • font-size: 字号，大小

  • font-weight: 字体粗细

    1. bold or 700：加粗
    2. bolder：重体
    3. lighter：细体
    4. 500：中体
    5. 100：最细体
    6. 900：最粗体

  • font-style: 字体倾斜

    1. italic：倾斜
    2. oblique：倾斜
    3. normal：正常

  • line-height: 行高，一行文字的高度（或者上行文字的顶部到下行文字的顶部）

    如果是数字，就是字高的倍数
    文字上下居中技巧: 设置 height 和 (line-height*行数) 一样高

  • font-family: 字体

    用“, ”隔开，浏览器会从左到右依次寻找，找到第一个存在的字体，就用这个字体
    最后一个用sans-serif无衬线字体
    一般从中文网页上直接复制该属性

    *{font-family: Microsoft YaHei, sans-serif; }
    设置字体为微软雅黑，如果找不到就使用无衬线字体

  • text-indent: 首行缩进

    数字+em，1em=1个字的像素数

    h5{text-indent: 2em;}
    设置h5标签的文本缩进为2个字

  • text-align: 文字对齐方式

    1. left：左对齐（默认）
    2. right：右对齐
    3. center：居中对齐

  • text-decoration: 修饰线

    一般用于超链接去除下划线
    box一般指定border-bottom
    1. none：无修饰线（默认）
    2. overline：上划线
    3. underline：下划线
    4. line-through：删除线

  • color: 字体颜色，背景色是background-color

    1. 颜色名写法：blue、red、yellow...
    2. rgb写法：rgb(0-255,0-255,0-255)
    3. rgba写法：rgba(0-255,0-255,0-255,0-1)
    4. 16进制写法：#rrggbb

  • font:是否倾斜 是否加粗 字号/行高 字体

    综合属性，将几个字体属性放在一起, 一般用于网页开发初期默认字体

  • text-shadow: 文字阴影

    阴影文字

  • -webkit-text-stroke: 网页套件文本描边

    描边文字
