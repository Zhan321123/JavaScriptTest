Attributes Modify

  1. element.style，标签样式

    element.innerHTML，标签内容

    element.innerText，标签内容，不包含标签

          const box = document.querySelector('#selector')
      box.innerHTML += ', strong'
      box.style.backgroundColor = 'red'
      box.style.border = '2px solid black'
    div id=selector

  2. element.className，获取class属性值，即"class1" or "class1 class2 ..."

    element.classList，标签class列表

          console.log(box.className)
      console.log(box.classList)
    div class="box2 box3"

  3. element.classList.add(className)，添加class属性值

    element.classList.remove(className)，删除class属性值

    element.classList.toggle(className)，切换class属性值，如果有就删除，没有就添加

    
      const d = document.querySelector('#div3')

      d.classList.add('box1')
      d.classList.add('box2')
      d.classList.remove('box2')

      d.onclick = function () {
        d.classList.toggle('box1')
      }
    div

  4. document.head.appendChild(document.createElement('style').innerHTML = "css样式")，写入css样式

          let style = document.createElement('style');
      style.innerHTML=`#div4{background-color: #679839;}`
      document.head.appendChild(style)
    div id=div4

  5. 自定义属性

    定义属性：data-属性名="属性值"

    获取属性：dataset.属性名

          console.log(d.dataset)
      console.log(d.dataset.id)
      console.log(d.dataset.icon)
    div data-id="1" data-icon="icon-16px">
