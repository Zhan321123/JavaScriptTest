Get and Create Elements

  1. document.querySelector("css选择器"/"#id")，获取单个元素，不存在返回null，存在多个时返回第一个匹配的元素

          document.querySelector("code")
      document.querySelector(".box")
      document.querySelector("#head")
      document.querySelector("input[type='text']")
    code

    p class=box


    input type=text

  2. document.querySelectorAll("css选择器")，获取多个元素，返回一个NodeList列表对象

          document.querySelectorAll("a")
    a a a

  3. document.getElementById("id")，获取id元素

    document.getElementsByClassName("class")，获取class元素

    document.getElementsByTagName("tag")，获取tag元素

    document.getElementsByName("name")，获取name元素

    document.getElementsByTagNameNS()，获取tag元素

          document.getElementById("btn")
      document.getElementsByClassName("box")
      document.getElementsByTagName("code")
      document.getElementsByName("password")
      document.getElementsByTagNameNS(svgNS, 'circle');

    p class=box

    code
    input name=password
    svg xmls="http://www.w3.org/2000/svg" fill="red"

  4. document.createElement("tag")，创建元素

    可以附带属性，比如div.box

          const div2 = document.getElementById('div2')
      const create = document.createElement('button')
      create.innerText = 'create button'
      div2.appendChild(create)
    div

  5. element.appendChild(元素)添加元素

    element.insertBefore(添加的元素, element之前)

    element.cloneNode(boolean)，复制元素，true时复制包括子孙元素，false只复制该标签，默认为false

    element.removeChild(子元素)删除子元素
