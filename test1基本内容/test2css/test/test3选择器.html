一般选择器

  • 普通选择器

    label {}
    所有该label的标签

    p{color:red}
    设置段落标签字体为红色

  • class选择器

    .class {}
    所有该class的标签
    使用多个class时，用空格分割class='class1 class2...'

    ep: .w{width:80%}
    设置class='w'的标签的宽度为80%

  • id选择器

    #id {}
    一个id的标签的样式，id不能给一个标签使用

    #output{font-size:20px}
    设置id="output"的标签的字体大小为20px

  • * 选择器

    * {}
    包含所有标签

    *{box-sizing:border-box}
    设置所有标签的边框模型为border-box

  • 反选选择器

    not(选择器){}，选择这种的选择器
    选择器1:not(选择器2){}，选择所有选择器1，但是匹配选择器2不会被选择

    ep: .box:not(span){}
    设置class="box"中所有不是span的标签的样式

复合选择器

  • 后代选择器

    label1 label2{}
    所有后代

    .box div{}
    设置class="box"中所有div的样式

  • 子代选择器

    label1 > label2{}
    只改变子代，孙代不改变

    .box > div{}
    设置class="box"中子代div的样式，.box div div 不会被选择

  • 并集选择器

    label1, label2, ...{}
    多个标签

    ol, ul{padding-left:20px}
    设置所有ol和ul的padding-left为20px

伪类选择器

  • 结构伪类选择器

    label: first-child，匹配第一个元素
    label: last-child，匹配最后一个元素
    label: nth-child(n)，匹配第n个元素，注意，从1开始！！！
    label: nth-child(f(n))，填好fn，n ∈ N，匹配所有值的元素
    用于匹配列表ul和ol种第n的元素

    ul:first-child{background-color:cyan}
    设置ul的第一个子标签的背景色为青色

  • 状态伪类选择器

    label: 状态{}
    hover: 鼠标悬停，走开取消
    active: 鼠标按下激活，松开取消
    focus: 获得焦点
    visited: 访问过
    link: 链接
    注意: 遵循link-visited-hover-active的顺序，否则无法生效
    工作中常用设置超链接的方法:
    a:{}
    a:hover{}

    button:hover{cursor:pointer}
    设置button标签的鼠标悬停样式为指针

  • 元素伪类选择器

    label::before{}，在元素前添加内容
    label::after{}，在元素后添加内容
    注意: 必须要有content属性，否则不会生效

    li::before{content:'+'}
    设置所有li标签前添加+号

继承属性

  • 默认情况下文字的属性直接继承父元素，除非自己有相应的样式
  • 配置两次相同属性时，不同的会叠加，相同的会选择后者生效
  • 属性特性的优先级：（范围越大，优先级越低）
    继承 < * < 直接标签 < class < id < 行内样式 < !important
  • 复合选择器的优先级过于复杂
